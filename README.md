# 🗑️ Smart Dustbin (Automatic Lid System)

An Arduino-based Smart Dustbin that automatically opens its lid when a hand or object comes near using an ultrasonic sensor and closes it automatically using a servo motor.

---

## ✨ Features

- 📏 Automatic distance detection using HC-SR04
- 🤖 Servo-controlled lid mechanism
- ⚡ Contactless operation for hygiene
- 🔄 Smooth distance averaging for stability
- 🛠️ Simple, low-cost automation project

---

## 🧠 Working Principle

The ultrasonic sensor continuously measures distance in front of the dustbin.

- If an object is detected within **50 cm**, the lid opens automatically.
- After a delay, the lid closes automatically.

---

## 🔧 Components Used

- Arduino UNO / Nano  
- HC-SR04 Ultrasonic Sensor  
- Servo Motor (SG90 / MG995)  
- LED (optional)  
- Jumper wires  
- Power supply / USB cable  

---

## ⚙️ Circuit Connections

| Component | Arduino Pin |
|----------|-------------|
| TRIG     | D7          |
| ECHO     | D6          |
| SERVO    | D5          |
| LED      | D10         |
